Futu_OpenD 目前提供两种安装运行方式，您可任选其一：

1. 可视化 Futu_OpenD：提供界面化应用程序，操作便捷，尤其适合入门用户
    对应文件夹：Futu_OpenD-GUI_版本号_系统名
    点击链接，了解安装详情：https://openapi.futunn.com/futu-api-doc/quick/opend-base.html

2. 命令行 Futu_OpenD：提供命令行执行程序，需自行进行配置，适合对命令行熟悉或长时间在服务器上挂机的用户
    对应文件夹：Futu_OpenD_版本号_系统名 
    点击链接，了解配置详情：https://openapi.futunn.com/futu-api-doc/opend/opend-cmd.html


Futu_OpenD provides two modes to run. You can choose one of them:
1. Visualisation Futu_OpenD: Provide user interface that is easy to use, especially for entry-level users.
    Corresponding folder: Futu_OpenD-GUI_version_platform 
    Click on the link for installation details: https://openapi.futunn.com/futu-api-doc/en/quick/opend-base.html

2. Command Line Futu_OpenD: Provide command line execution program that needs to be configured by yourself. Suitable for users who are familiar with the command line or hang on the server for a long time.
    Corresponding folder: Futu_OpenD_version_platform 
    Click on the link for configuration details: https://openapi.futunn.com/futu-api-doc/en/opend/opend-cmd.html
